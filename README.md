“AI-Assisted Task Manager (Mini Portfolio Project)”

Project Overview:

This project is a full-stack web application built using a React + Vite frontend and a Node.js + Express backend.
The frontend provides a fast and responsive user interface, while the backend handles API requests, server-side logic, and communication with the database.
The goal of the project is to demonstrate a clean separation between client-side and server-side architecture, making the application scalable, maintainable, and easy to extend.

Key features include:
Modular and reusable frontend components
Clean folder organization
Easy setup for development

Steps to Run Frontend:
✅ 1. Navigate to the frontend directory
cd frontend

✅ 2. Install dependencies
npm install

✅ 3. Start the development server
npm run dev

✅ 4. Open the project in your browser
URL: http://localhost:5173

Steps to Run Backend:
✅ 1. Navigate to the backend directory
cd backend

✅ 2. Install dependencies
npm install

✅ 3. Start the backend server
For nodemon: npm run dev

✅ 4. Backend server runs at:
http://localhost:4000
